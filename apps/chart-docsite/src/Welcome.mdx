# Fluent UI Contributors Packages

Welcome to the documentation site for Fluent UI Contributors packages! These packages are published under the `@fluentui-contrib/` scope and provide additional functionality to enhance your Fluent UI experience.

## Available Packages

Explore the various packages available to extend your Fluent UI capabilities:

- [**houdini-utils**](/docs/packages-houdini-utils--docs): Utilities for CSS Houdini.

- [**react-chat**](/docs/packages-react-chat-chat--docs): React component for chat interfaces.

- [**react-data-grid-react-window**](/docs/packages-react-data-grid-react-window--docs): DataGrid with vertical virtualization using react-window.

- [**react-data-grid-react-window-grid**](/docs/packages-react-data-grid-react-window-grid--docs): DataGrid with 2D virtualization using react-window.

- [**react-draggable-dialog**](/docs/packages-react-draggable-dialog--docs): Draggable dialog component.

- [**react-headless-provider**](/docs/packages-react-headless-provider--docs): Provides a replacement for `FluentProvider` that does not render additional DOM elements.

- [**react-interactive-tab**](/docs/packages-react-interactive-tab--docs): InteractiveTab component enables you to include interactive elements, such as buttons, within a tab.

- [**react-resize-handle**](/docs/packages-react-resize-handle--docs): Component for creating resizable layouts.

- [**react-shadow**](/docs/packages-react-shadow--docs): Adapter for using Fluent UI with `react-shadow`.

- [**react-tree-grid**](/docs/packages-react-tree-grid--docs): TreeGrid component for hierarchical data.

## Getting Started

To get started with any of these packages in your project:

1. Install the desired package using `npm` or `yarn`:

   ```
   npm install @fluentui-contrib/<package-name>
   ```

   or

   ```
   yarn add @fluentui-contrib/<package-name>
   ```

2. Import and use the components in your React application.

3. Refer to the individual package documentation (Storybook links above) for detailed usage instructions and examples.
