The `@fluentui/react-icons` package lets you use the whole [Fluent UI System Icons](https://github.com/microsoft/fluentui-system-icons) icon set. Each icon is wrapped in a React component, using SVG by default for maximum sharpness, performance and flexibility.

## System Icons

This is Fluent's richest icon catalog, available in two styles: `Filled` and `Regular`. Use **Unsized** icon IDs (eg. `SendRegular` or `SendFilled`, rendered at **1em × 1em** by default) and can be adjusted proportionally using the `fontSize` property.

**Sized** icons (eg. `Send24Regular` or `Send32Regular`) include their base width/height in their name, and are optimized to look crisp at predetermined sizes. You should favor using the **unsized** icons to give your app a performance gain by reusing assets, but if the icon has logos or details that don't scale effectively, use the ones with the size in their name.

## Emojis

As a general guideline, it's recommended to utilize unsized icons since the same instance of an icon can be reused multiple times. However, there are instances where icons may not scale effectively. For example, icons containing logos and signs often have varied glyph sizes. In such cases, it's advisable to opt for sized icons.
