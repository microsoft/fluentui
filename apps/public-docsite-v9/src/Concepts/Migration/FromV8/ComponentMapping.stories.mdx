# Getting Started

1. **Install Fluent UI v9**: 
   To begin migration, install the latest Fluent UI React v9 package.

   ```bash
   npm install @fluentui/react-components
   ```

2. **Update Imports**: 
   Update your component imports to use the v9 components. Use the component mapping table below to assist.

3. **Refactor Code**: 
   Refactor your code to accommodate changes in API, props, and new functionalities provided by v9 components.

# New Components in v9
React v9 introduces several new components that were not present in v8:
1. [**Avatar**](https://react.fluentui.dev/?path=/docs/components-avatar--docs): Replaces `Persona` for rendering user images.
2. [**Popover**](https://react.fluentui.dev/?path=/docs/components-popover--docs): Enhanced version of `Callout` with improved accessibility.
3. [**Drawer**](https://react.fluentui.dev/?path=/docs/components-drawer--docs): Replaces `Panel` for side-drawer navigation.
4. [**TabList**](https://react.fluentui.dev/?path=/docs/components-tablist--docs): Replaces `Pivot` for tabbed navigation.
5. [**ProgressBar**](https://react.fluentui.dev/?path=/docs/components-progressbar--docs): Renamed from `ProgressIndicator`, with similar functionality but refined API.
6. [**Input**](https://react.fluentui.dev/?path=/docs/components-input--docs): A new input component replacing `TextField` for simpler text input handling.
7. [**Switch**](https://react.fluentui.dev/?path=/docs/components-switch--docs): A new component replacing `Toggle` for binary options.
8. [**SwatchPicker**](https://react.fluentui.dev/?path=/docs/components-swatchpicker--docs): A new component replacing `SwatchColorPicker` with added functionality.
9. [**Accordion**](https://react.fluentui.dev/?path=/docs/components-accordion--docs): Allows users to toggle the display of content by expanding or collapsing sections.
10. [**Badge**](https://react.fluentui.dev/?path=/docs/components-badge-badge--docs): Displays text and/or an icon.
11. [**CounterBadge**](https://react.fluentui.dev/?path=/docs/components-badge-counter-badge--docs): Displays Badge numerical values.
12. [**PresenceBadge**](https://react.fluentui.dev/?path=/docs/components-badge-presencebadge--docs): Displays Badge status.
13. [**InfoLabel**](https://react.fluentui.dev/?path=/docs/components-badge-presencebadge--docs):An InfoLabel is a Label with an InfoButton at the end.
14. [**Toast**](https://react.fluentui.dev/?path=/docs/components-badge-presencebadge--docs):A Toasts displays temporary content to the user.
15. [**TeachingPopover**](https://react.fluentui.dev/?path=/docs/components-teachingpopover--docs): A new component in v9 that replaces `TeachingBubble` and `Coachmark` in v8.
16. [**Carousel**](https://react.fluentui.dev/?path=/docs/components-carousel--docs): Sliding window of elements controlled by previous, next, and direct pagination buttons.
