# Migration Shims/V8/CheckboxShim

The CheckboxShim component accepts the same props as a v8 Checkbox and renders a v9 Checkbox.

## Props

| Name                   | Type                                                           | Required                                       | Default    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ---------------------- | -------------------------------------------------------------- | ---------------------------------------------- | ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| `componentRef`         | `IRefObject<ICheckbox>`                                        | No                                             |            | Optional callback to access the `ICheckbox` interface. Use this instead of `ref` for accessing the public methods and properties of the component.                                                                                                                                                                                                                                                                                                                                           |
| `className`            | `string`                                                       | No                                             |            | Class name to provide on the root element, in addition to the `ms-Checkbox` class.                                                                                                                                                                                                                                                                                                                                                                                                           |
| `checked`              | `boolean`                                                      | No                                             |            | Checked state. Mutually exclusive with `defaultChecked`. Use this if you control the checked state at a higher level and plan to pass in the correct value based on handling `onChange` events and re-rendering.                                                                                                                                                                                                                                                                             |
| `defaultChecked`       | `boolean`                                                      | No                                             |            | Default checked state. Mutually exclusive with `checked`. Use this if you want an uncontrolled component, meaning the Checkbox instance maintains its own state.                                                                                                                                                                                                                                                                                                                             |
| `label`                | `string`                                                       | No                                             |            | Label to display next to the checkbox.                                                                                                                                                                                                                                                                                                                                                                                                                                                       |

| `title`                | `string`                                                       | No                                             |            | Title text applied to the root element and the hidden checkbox input. (Use `label` instead for the visible label.)                                                                                                                                                                                                                                                                                                                                                                           |
| `id`                   | `string`                                                       | No                                             |            | ID for the checkbox input.                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| `name`                 | `string`                                                       | No                                             |            | Name for the checkbox input. This is intended for use with forms and NOT displayed in the UI.                                                                                                                                                                                                                                                                                                                                                                                                |

Including `data-*` props in `inputProps` is supported but currently requires casting since TS 3.7 doesn't provide a way to allow all keys with a certain prefix. |
| `boxSide`              | `"start" "end"`                                                | No                                             |            | Determines whether the checkbox should be shown before the label (`start`) or after (`end`). @defaultvalue 'start'                                                                                                                                                                                                                                                                                                                                                                           |

| `ariaLabel`            | `string`                                                       | No                                             |            | Accessible label for the checkbox.                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `ariaLabelledBy`       | `string`                                                       | No                                             |            | ID for element that contains label information for the checkbox.                                                                                                                                                                                                                                                                                                                                                                                                                             |

| `indeterminate`        | `boolean`                                                      | No                                             |            | Optional controlled indeterminate visual state for checkbox. Setting indeterminate state takes visual precedence over `checked` or `defaultChecked` props given but does not affect checked state. This should not be a toggleable state. On load, the checkbox will receive indeterminate visual state. After the first user click, your supplied `onChange` callback should remove the indeterminate state (without modifying the checked state), exposing the true state of the checkbox. |
| `defaultIndeterminate` | `boolean`                                                      | No                                             |            | Optional uncontrolled indeterminate visual state for checkbox. Setting indeterminate state takes visual precedence over `checked` or `defaultChecked` props given but does not affect checked state. This should not be a toggleable state. On load, the checkbox will receive indeterminate visual state. After the user's first click, it will be removed, exposing the true state of the checkbox.                                                                                        |
