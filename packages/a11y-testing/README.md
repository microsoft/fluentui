# @fluentui/a11y-testing

Tests for testing a11y conformance of components and hooks, together with typings for creating custom definitions.

> [!NOTE]
> This package was developed and used primarily for react-northstar (v0).
>
> There are two usages within v9 tests
>
> - [`react-button` package](./src/definitions/react-button/buttonAccessibilityBehaviorDefinition.ts)
> - [`react-link` package](./src/definitions/Link/linkBehaviorDefinition.ts)
