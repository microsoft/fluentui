# @fluentui/react-file-type-icons-stories

Storybook stories for packages/react-file-type-icons

## Usage

To include within storybook specify stories globs:

```js
module.exports = {
  stories: [
    '../packages/react-file-type-icons/stories/src/**/*.stories.mdx',
    '../packages/react-file-type-icons/stories/src/**/index.stories.@(ts|tsx)',
  ],
};
```

## API

No public API available.
