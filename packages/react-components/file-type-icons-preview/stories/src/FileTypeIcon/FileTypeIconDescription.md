# FileTypeIcon

FileTypeIcon displays an icon representing a file type based on its extension or a special type (like folder).

The component automatically selects the appropriate icon from the Fluent Design file type icon set and handles device pixel ratio for optimal display quality.

## Features

- **Automatic icon selection**: Matches file extensions to the appropriate icon
- **Multiple sizes**: Supports 16, 20, 24, 32, 40, 48, 64, and 96 pixel sizes
- **Format support**: Renders icons as SVG or PNG
- **Special types**: Supports non-file-based icons like folders, list items, etc.
- **Accessibility**: Includes appropriate alt text for screen readers
