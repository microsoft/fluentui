`overflowBoundary` names a rect the positioned surface must stay within. When set, the hook writes logical `max-inline-size` and `max-block-size` on the surface so its far edge can never extend past the boundary — the surface's start edge stays anchored where CSS anchor positioning placed it, and content reflows inside the clamped box. Pair with `overflowBoundaryPadding` to reserve breathing room between the surface and the boundary edges.
