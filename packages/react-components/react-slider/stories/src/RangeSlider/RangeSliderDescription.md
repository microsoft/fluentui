# RangeSlider

The RangeSlider exposes the same visual treatment as the single-value Slider but allows people to select both a minimum and maximum value along the rail. Each thumb can be dragged with the mouse, touched, or adjusted with the keyboard, and the component renders two visually hidden (screen-reader only) native inputs so both values participate in form submissions without recreating slider semantics. Use it when a filter or control needs a bounded range rather than a single point.
